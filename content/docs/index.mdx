Welcome to the NextNest documentation. This is where you can find all the information you need to get started with our premium starter kit.

## What is NextNest?

NextNest is a premium Next.js & NestJS starter kit designed for speed, security, and developer joy.

### Key Features

- **End-to-end Type Safety**: Shared types between frontend and backend.
- **Modern Auth**: Pre-configured BetterAuth integration.
- **Database Ready**: Prisma ORM with optimized configurations.
- **Premium UI**: Styled with Tailwind CSS 4 and Fumadocs.
