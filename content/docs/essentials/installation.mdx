Follow these steps to set up NextNest on your local machine.

## Prerequisites

Before you begin, ensure you have the following installed:

- Node.js 20+
- Bun or NPM
- PostgreSQL database

## Setup Steps

### 1. Clone the repository

```bash
git clone https://github.com/your-repo/nextnest.git
cd nextnest
```

### 2. Install dependencies

```bash
bun install
```

### 3. Configure environment variables

Copy the `.env.example` file to `.env` and fill in your details.

```bash
cp .env.example .env
```

### 4. Run database migrations

```bash
bun db:migrate
```

### 5. Start development server

```bash
bun dev
```
