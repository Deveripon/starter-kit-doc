NextNest uses Prisma as its primary ORM for type-safe database access.

## Schema Definition

Define your database schema in `prisma/schema.prisma`.

```prisma
model User {
  id    Int     @id @default(autoincrement())
  email String  @unique
  name  String?
  posts Post[]
}

model Post {
  id        Int     @id @default(autoincrement())
  title     String
  content   String?
  published Boolean @default(false)
  author    User    @relation(fields: [authorId], references: [id])
  authorId  Int
}
```

## Useful Commands

### Generate Client

```bash
bun prisma generate
```

### Studio

```bash
bun prisma studio
```

### Push Changes

```bash
bun prisma db push
```
