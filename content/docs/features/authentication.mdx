NextNest comes with BetterAuth pre-configured out of the box.

## Configuration

You can find the authentication configuration in `lib/auth.ts`.

### Social Logins

To enable social logins, add the respective client IDs and secrets to your `.env` file.

- `GITHUB_CLIENT_ID`
- `GITHUB_CLIENT_SECRET`
- `GOOGLE_CLIENT_ID`
- `GOOGLE_CLIENT_SECRET`

## Protect Routes

Use the `auth` middleware to protect your Next.js routes.

```tsx
import { auth } from "@/lib/auth";

export default async function Dashboard() {
  const session = await auth();
  
  if (!session) {
    return <div>Access Denied</div>;
  }

  return <div>Welcome, {session.user.name}</div>;
}
```
